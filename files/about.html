Retail Management System


This is a Retail Management System.
It can facilitate easy generation, viewing and management of any data relevant for a reatil store.
It uses HTML and CSS in the client side and Node frameword of JavaScript in the server side.
It has 2 types of logins, namely, User and Admin. The User can generate new bills for any product available. The Admin can view all the bill generated so far with all the details of the bills
